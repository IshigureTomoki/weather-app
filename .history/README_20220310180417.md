# 天気予報アプリケーション

Reactの基礎学習のために、天気予報を表示するWEBアプリケーションを作成。

## デモ

## 機能概要
- ページ表示時に現在地を取得
- 位置情報に基づき、現在・1週間・24時間（グラフ）の天気とマップを表示
- 任意の地域を検索可能


## 使い方

1. このリポジトリをクローンします。

```bash
$ git clone https://github.com/IshigureTomoki/weather-app
```

2. Go into the repositor

```bash
$ cd weather-app
```

3. Install dependencies

```bash
$ yarn
```

4. Run the app

```bash
$ yarn start
```
